Урок 2: Об'єктно-орієнтоване програмування у Java

ООП (об'єктно-орієнтоване програмування) — це підхід, у якому програма будується навколо об'єктів. Кожен об'єкт — це комбінація даних (полів) і дій над ними (методів).

Java є повністю об'єктно-орієнтованою мовою, де класи — основа всього. Правильне розуміння ООП допоможе тобі писати чистий, зрозумілий і повторно використовуваний код.

Чотири основи ООП

  • Інкапсуляція — приховування деталей реалізації та доступ через публічні методи.
  • Наслідування — створення нового класу на основі вже існуючого.
  • Поліморфізм — можливість одного методу вести себе по-різному залежно від контексту.
  • Абстракція — виділення суттєвого, приховування зайвого.

Клас та об'єкт у Java

Клас — це шаблон, а об'єкт — конкретний екземпляр цього шаблону.

Ось приклад простого класу:


public class Person {
      private String name;
      private int age;

      // Конструктор
      public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Метод
    public void sayHello() {
        System.out.println("Привіт, мене звати " + name);
    }
}

А ось як створити об'єкт цього класу:


Person user = new Person("Олена", 22);
user.sayHello(); // Виведе: Привіт, мене звати Олена

Ключові терміни

  • private — обмежує доступ до поля або методу лише в межах класу.
  • public — дозволяє доступ з будь-якого місця.
  • this — посилання на поточний об'єкт.
  • new — створює новий об'єкт.
