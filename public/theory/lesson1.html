Урок 1: Вступ до Java

Java — це мова програмування загального призначення, створена компанією Sun Microsystems у 1995 році. Вона широко використовується для створення додатків: від мобільних до корпоративних. Java працює за принципом «Write Once, Run Anywhere», що означає, що код, написаний один раз, може працювати на будь-якому пристрої з встановленою JVM.

Переваги Java

  • Проста у вивченні для початківців
  • Об'єктно-орієнтована (OOP)
  • Безпечна та надійна
  • Платформонезалежна
  • Має велику стандартну бібліотеку

Перша програма на Java

Найпростіша Java-програма виглядає так:


public class HelloWorld {
      public static void main(String[] args) {
        System.out.println("Привіт, Java!");
    }
}

Пояснення

  • public class HelloWorld — створюємо клас з іменем HelloWorld.
  • public static void main(String[] args) — точка входу в програму.
  • System.out.println(...) — виводить текст у консоль.

Основні поняття Java

  • Клас — шаблон для створення об’єктів.
  • Метод — блок коду, який виконує певну дію.
  • Змінна — контейнер для зберігання даних.
  • Типи даних — int, double, boolean, String тощо.

Корисні поради

  • Завжди починай назви класів з великої літери.
  • Форматуй код, щоб його було легко читати.
  • Вивчай приклади і змінюй їх — так краще засвоїш матеріал.
  • Не бійся помилок — вони допомагають навчитися.
