Урок: Сортування бульбашкою

Сортування бульбашкою — це базовий алгоритм, який багато хто вивчає на початку. Він простий для розуміння, хоча й не найшвидший. Його суть: послідовно порівнювати сусідні елементи масиву та міняти їх місцями, якщо порядок неправильний.

Принцип роботи

  • Беремо два сусідні елементи.
  • Якщо перший більший за другий — міняємо місцями.
  • Повторюємо це до кінця масиву, і знову, поки все не буде впорядковано.

Візуалізація

Код на Java


public class BubbleSort {
      public static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
      for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
      // Обмін
      int temp = arr[j];
      arr[j] = arr[j + 1];
      arr[j + 1] = temp;
                }
            }
        }
    }

    public static void main(String[] args) {
        int[] numbers = {5, 2, 9, 1, 3};
        bubbleSort(numbers);
        for (int num : numbers) {
      System.out.print(num + " ");
        }
    }
}

Що тут відбувається?

  • Метод bubbleSort реалізує сортування.
  • Зовнішній цикл — кількість проходів по масиву.
  • Внутрішній цикл — порівняння сусідів і обмін.
  • У main — створення масиву, виклик сортування і вивід результату.

Недоліки та застосування

  • Повільний для великих обсягів даних: складність O(n²).
  • Добре підходить для навчання, але у реальних задачах його рідко використовують.
  • Можна оптимізувати: якщо за один прохід не було жодного обміну — завершити достроково.
